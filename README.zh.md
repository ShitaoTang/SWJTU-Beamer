# 西南交通大学计算机系 Beamer 演示模板

[English Version](./README.md)

这是一个基于 LaTeX Beamer 的演示模板，适用于 **西南交通大学计算机系** 的学术演示。  
模板风格参考了 [Oxford Mathematical Institute Beamer Template](https://www.overleaf.com/latex/templates/oxford-mathematical-institute-beamer-presentation/zxkbrjpszxjr)。

---

## 关于色彩标准

本模板参考了  
[西南交通大学视觉识别系统（VIS）](https://news.swjtu.edu.cn/gywm/zlxz.htm)。

- 官方标准色 **“交大蓝”** 定义为 **C90 M40 Y5 K5**。  
- 本模板采用的 RGB 值来自 *《西南交通大学视觉识别系统（VIS）手册（矢量文件）》(2019.06.03 发布)*，在 Macbook 上通过数码测色计测得。  
- 最终 RGB 值为 **(51, 117, 174)**。  
  （由于公式换算得到的 RGB 在电子分发版上的视觉效果不佳，因此使用了实测值。）
